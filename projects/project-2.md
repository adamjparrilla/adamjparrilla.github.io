During the spring semester of 2020, I was in the course ICS 211 where we began learning about the basics of recursion and slightly more complex Computer Science ideas. In this project, with the help of Cam Moore I implemented Java code that can recursively backtrack to solve a hexadecimal Sudoku problem. 

In doing this project I gained experience working with more complex ideas, such as boolean's, matrices and using test classes, while also further helping me to gain more knowledge about programming.
