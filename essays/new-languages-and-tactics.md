## New JavaScript language

As a new programmer to JavaScript I can see the appeal of this language because it has much more flexibility. For example, the fact that arrays don’t need to contain all of the same type of objects, allows for a much more straight forward and simple code in comparison to Java or C++. It also has first class functions, meaning unlike the difficult to manipulate functions in Java and C++, you can do things like assign functions to another function or create collections of functions. Which just allows you to generate functions more dynamically than you could in the other languages. Personally, I like JavaScript, so far we have only learned the basics and will continue to learn more as we move along. Although there is a few differences from other languages I’ve used before such as, destructuring or arrow functions, it seems to be a very easy to use language that is similar to Java and C++. 

## New Teaching Techniques

Athletic software engineering is an interesting teaching tactic and as of now, I do think that it is a bit stressful learning this way. But, I’m trying to continue to push myself to get accustomed to this way of learning however, and hopefully it will allow me to have the most productive semester I’ve ever had. I do think that the WODs(workout of the day) are helpful especially because I have ample opportunity to go in and seek help when I need it. Its also is very helpful to have a dedicated time to practice problem solving and critical thinking like this. I’m not exactly sure how this style of learning will do for me as of now, but I’m hoping that it will push me to gain more competency than I have before.
