## New JavaScript language

As a new programmer to JavaScript, I can see the appeal of this language because it has so much flexibility. For example, the fact that arrays don’t need to have all the same type of objects, allows for a much more straight forward and simple code in comparison to Java or C++. It also has first class functions, meaning unlike the difficult to manipulate functions in Java and C++, you can do things like assign functions to another function or create collections of functions. Which just allows you to generate functions more dynamically than you could in the other languages. Personally, I like JavaScript because as an relatively inexperienced programmer its very simple to use and straight forward. So far we've only learned some of the basics, but as we move along it will be interesting to see how much more I can do with this language. Some other very cool things that I didn't know you could do were things such as destructuring and arrow functions. I personally intend on investing time and effort into learning this to the best of my ability so that I can hopefully improve my competency in coding. 


## New Teaching Techniques

I think that athletic software engineering is an interesting teaching tactic. Although it is a bit stressful learning this way, I do think it will put the pressure on myself to putting in the extra time needed to be successful in the course and career path. I'm going to spend time everyday to improve my code, and hopefully that will result in the outcome of a good grade. The workout of the day or WOD's that we do are especially helpful for me, and even though I've never been great under pressure its teaching me to be calm and think one step at a time until the goal is reached. ICS 314 has also been nice so far because whenever I was confused or needed help I was always able to ask someone whether that be my instructor or TA. In the end I'm not exactly sure what this new athletic software engineering learning style will do for me, but I'm hoping that giving it enough time and dedication I can gain more knowledge than I have ever before. 

## New Experiences

Overall I'm excited to learn more this semester and after my first WOD today, it was exactly what I assumed it would be. Stressful, and a bit scary but with that, I also think it was important for me to do. Challenges will continuously come up in this course and in this career so I think its important that I learn how to prepare for questions like these that might be asked in an interview. Finally, I think learning Javascript and taking this class is going to make me a better problem solver and overall computer scientist, so I'm going to work hard no matter what challenges there may be.
